## 요구사항

1. **랜딩페이지**
   - 미니멀 디자인
   - 동영상/GIF 히어로 섹션 (플레이스홀더로 시작)
   - 배경음악 온/오프 토글
   - SEO 최적화 (메타태그, 구조화 데이터)

2. **블로그 기능**
   - 메뉴로 글 탐색
   - 카테고리별 분류
   - 원본 작성 날짜 유지

## 구현 단계

### Phase 1: 프로젝트 초기화
```
/Users/joejeon/develop/유자살롱 홈피 복원/
```

1. Astro 프로젝트 생성
2. 기본 설정 (astro.config.mjs)
3. GitHub Pages 배포 설정

### Phase 2: 레이아웃 및 컴포넌트
생성할 파일:
- `src/layouts/BaseLayout.astro` - 공통 레이아웃, SEO
- `src/components/Navigation.astro` - 메뉴
- `src/components/MusicPlayer.astro` - 배경음악 플레이어
- `src/components/Hero.astro` - 동영상/GIF 섹션
- `src/components/Footer.astro` - 푸터

### Phase 3: 페이지 구조
- `src/pages/index.astro` - 랜딩페이지
- `src/pages/blog/index.astro` - 글 목록
- `src/pages/blog/[...slug].astro` - 개별 글
- `src/pages/category/[category].astro` - 카테고리별 글

### Phase 4: 콘텐츠 마이그레이션 스크립트
`scripts/migrate.js` 작성:

1. **HTML 파싱**
   - 제목: `<h2 class="title-article">`
   - 카테고리: `<p class="category">`
   - 날짜: `<p class="date">`
   - 태그: `<div class="tags">`
   - 본문: `<div class="contents_style">`

2. **마크다운 변환**
   - HTML → Markdown (turndown 라이브러리)
   - 이미지 경로 수정: `./img/` → `/images/posts/{id}/`

3. **파일 생성**
   ```
   src/content/posts/
   ├── 5-두줄기타를-소개합니다.md
   ├── 100-공연영상-2010-창의서밋.md
   └── ...
   ```

4. **이미지 복사**
   ```
   public/images/posts/
   ├── 5/
   │   └── 2222.jpg
   ├── 100/
   └── ...
   ```

### Phase 5: 스타일링
- `src/styles/global.css` - 미니멀 디자인 시스템
- 반응형 레이아웃
- 다크/라이트 모드 (선택)

### Phase 6: 배포
1. GitHub 저장소 생성 (gogodevelop2/yoojasalon_blog)
2. GitHub Actions 워크플로우 설정
3. 커스텀 도메인 연결 (yoojasalon.net)
